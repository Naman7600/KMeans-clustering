# KMeans-clustering

This project uses the K-Means Clustering algorithm to group cricket players based on their performance metrics. It helps in identifying similar player profiles using unsupervised machine learning.

## 🚀 Features

* K-Means clustering implementation
* Data preprocessing and cleaning
* Visualization of clusters
* Performance-based grouping of players

## 📂 Dataset

* Dataset: cricket.csv
* Contains player statistics such as runs, strike rate, averages, etc.

## ▶️ Usage

python src/kmeans.py
jupyter notebook notebooks/analysis.ipynb

## 📈 Output

* Clustered players based on performance
* Visual plots showing clusters
* Insights into player categories

## 🧠 How It Works

1. Load and preprocess cricket dataset
2. Select relevant features
3. Apply K-Means clustering
4. Assign players to clusters
5. Visualize results

## 🛠️ Tech Stack

* Python
* NumPy
* Pandas
* Matplotlib / Seaborn
* Scikit-learn

* Player performance analysis
* Sports analytics
